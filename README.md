# How to build
* Install gradle and JDK 8
* Clone this repo
* Open cmd or bash or something what you like
* Go to cloned folder
* Run two commands:
  * `gradle setupDevWorkspace`
  * `gradle build`
* Jar file would be in `.\build\libs` folder
